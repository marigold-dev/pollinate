Module Common.Tqueue

Thread-safe queues with the same exact interface as OCaml's Queue module, except any function that reads from or writes to a Queue returns a promise. Furthermore, there is a special take function for blocking the current thread until an element is available to take from the queue. Read the documentation for OCaml's Queue module for more information about the functions in this module

type 'a t

The type of a thread-safe queue

Blocks the current thread of execution until an element is in the queue, then returns a promise containing the element
