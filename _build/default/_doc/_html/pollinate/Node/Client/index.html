Module Node.Client

Client operations on a node in a P2P application

val address_of : 'a Node__Types.node -> Common.Address.t

Retrieve the Address.t of the given node

val peer_from : 'a Node__Types.node -> Common.Peer.t

Constructs a Peer.t from a node

val add_peer : 'a Node__Types.node -> Common.Peer.t -> [ `Duplicate | `Ok ]

Add a peer to the know peers

val create_request : 'a Node__Types.node Stdlib.ref -> Common.Address.t -> bytes -> Message.t Lwt.t

`create_request node recipient payload` creates a Message.t of the Request category addressed to `recipient` containing `payload`.

val create_response : 'a Node__Types.node Stdlib.ref -> Message.t -> bytes -> Message.t

`create_response node request payload` creates a Message.t of the Response category that responds to `request` whose content is `payload`.

val send_to : 'a Node__Types.node Stdlib.ref -> Message.t -> unit Lwt.t

Sends a message via datagram from the given node to a specified peer. Construct a message with one of the create functions to then feed to this function.

val recv_next : 'a Node__Types.node Stdlib.ref -> Message.t Lwt.t

Waits for the next incoming message and returns it

val request : 'a Node__Types.node Stdlib.ref -> bytes -> Common.Address.t -> Message.t Lwt.t

Sends an encoded request to the specified peer and returns a promise holding the response from the peer. This function blocks the current thread of execution until a response arrives.

val broadcast_request : 'a Node__Types.node Stdlib.ref -> bytes -> Common.Address.t list -> Message.t Lwt.t list

Broadcasts a request containing the given payload to a list of recipients and collects the responses in a list of `Message.t Lwt.t`.
