Represents a node with some state in a peer-to-peer network

val init : state:'a -> ?router:( Message.t -> Message.t ) -> msg_handler:( 'a Stdlib.ref -> Message.t -> bytes ) -> ?init_peers:Common.Address.t list -> (string * int) -> 'a t Stdlib.ref Lwt.t

Initializes the node with an initial state, an optional routing function that the consumer can use to inspect and modify the incoming message as well as its metadata, and a message handler that acts on the current state and the Message.t representing the request. The message handler is used to initialize a server that runs asynchronously. Returns a reference to the node.
